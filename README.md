# SuperHide

**SuperHide** is a lightweight Paper/Spigot plugin that allows players with permission to completely hide themselves from the player list (TAB).

Unlike some plugins, SuperHide does **not** leave empty or glitched nicknames - the player is fully removed from the player list.
